## Global Superstore Sales Analysis

## Project Overview

This project analyzes the Global Superstore dataset using **Python, MySQL, and Power BI**. The objective is to clean the data, perform exploratory data analysis (EDA), generate business insights using SQL queries, and build an interactive dashboard for decision-making.

- Pandas
- NumPy

- Power BI
